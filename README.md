# Data Structures and Algorithms

A collection of core Data Structures and Algorithms (DSA) implementations in Java, organized by topic for structured learning and practice.

### Queues

- Circular Queue  

### Searching
- Linear Search  
- Binary Search  

### Sorting
- Bubble Sort  
- Selection Sort  
- Insertion Sort
